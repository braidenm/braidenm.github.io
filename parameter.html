Simple Explanation of an Argument vs Parameter

Home Page

Lets keep it pretty simple then I will add some examples. Inside of a method (like a function) you have the parameter and the argument. The Parameter is a variable that is defined in another method. The argument is what the method returns. Ok, it is better if I show you:

First we need a class and a main (method)

Public class Example{

Public static void main(String[ ] args) {

Int sum = method(2, 5) // the argument being returned here

system.out.println(sum); //the result of the argument is stored in the int “sum” and displayed

}

Public static void method( int a, int b) {

//in the ( ) is the parameter. A and b are correlated with 2 and 5. The name of the method here is “method”

Return a + b; //this would be the argument returning the sum of integers a + b

}

}

Not much more to it than that.
